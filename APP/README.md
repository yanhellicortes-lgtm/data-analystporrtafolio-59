El objetivo es realizar una aplicación web Streamlit basada en un análisis exploratorio de datos (EDA) de anuncios de vehiculos. Brindando una forma sencilla e intuitiva de explorar los datos y visualizar patrones importantes de vehículos.
La aplicación tiene:
- Tabla completa del dataset.
- Histograma interactivo de la columna `odometer`.
- Gráfico de dispersión entre `odometer` y `price`.
- Filtraciones de las  visualizaciones según selección del usuario.
